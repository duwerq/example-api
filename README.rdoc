== example-api

This is a ruby on rails app that accepts a website address, indexes the h1, h2, h3 header tag along with all website links and displays the content.

== Running Locally

    git clone git@github.com:twolips/example-api.git
    cd example-api
    bundle install
    rake db:create db:migrate
    rails s

Your apps shouuld now be running on localhost:3000
